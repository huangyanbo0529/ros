实践描述：
亲爱的慕友们，在我们学习了 C++ 的引用、默认参数和类型安全等基础特性后，是不是感受到它比 C 语言更简洁、更安全？这些看似微小的改进，恰恰是 ROS2 节点 API 设计优雅的关键！现在，让我们通过一个贴近机器人开发场景的小项目，夯实函数设计的基础能力。欢迎小伙伴一起讨论：为什么现代 C++ 接口要避免裸指针和可变参数？
题目：
在 ROS2 节点初始化阶段，常需为传感器设置采样率、量程、日志开关等参数。为提升 API 友好性，请设计一个支持灵活调用的初始化函数，体现 C++ 相比 C 的简洁性与安全性。
输入与预期输出

void initSensor(
    const std::string& name,
    int rate = 10,
    double range = 5.0,
    bool logEnabled = false
);
